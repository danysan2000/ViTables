Development

If you are interested in how the ViTables development is going on you can visit its project site. From there you can:

  • follow and report bugs using the bug tracker

  • browse the source code repository via web

  • view project activity statistics

Feature Requests, Comments and Kudos

Your feedback is important! I'd really like to hear what you are doing with ViTables. It is the best way to adapt it to your needs. If you have feature requests, comments or kudos, please send a message to the ViTables Users Group.
